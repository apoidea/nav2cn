# nav2 中文网

简要描述项目背景以及功能。

# 运行环境

- Node.js 版本 14.17.0 以上
- 全局 npm 版本保持最新
- 现代浏览器和 IE11 以上

# 本地开发

1. 进入项目根目录，执行 `npm run install-dep` 命令安装依赖包；

3. 执行 `npm run dev` 命令启动项目；
