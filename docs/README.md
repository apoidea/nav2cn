## Nav2

Nav2继承于ROS导航栈。Nav2旨在找到一种让机器人从A点安全的移动到B点的方法。它也可以应用于涉及机器人导航的其他应用，例如跟路点跟踪。这将完成动态路径规划、计算电机速度、动态避障和恢复行为

### 开源硬件平台fishbot
由小鱼完成初步设计，fishbot开源小组完善的开源机器人平台，包括仿真和实体机器人。[项目详情](pages/sources/fishbot/README.md)
