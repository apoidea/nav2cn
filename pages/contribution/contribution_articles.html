# 本站如何在本地跑起来？

本站的最新版本使用的是 VuePressopen in new window 文档生成工具来驱动的。

由于 VuePressopen in new window 是基于 Nodejsopen in new window 编写的工具，如果你想让文档在本地运行调试，你首先需要安装 Nodejsopen in new window

  • Windows: Windows操作系统的小伙伴可以直接下载最新版本的 Nodejs 的 Windows 安装包open in new window
  • 非Windows: 非Windows操作系统推荐使用 nvmopen in new window 来安装 Nodejsopen in new window
